ConEmu 使用简介

发表于 2018-11-02 | 分类于 开发工具

Linux 系统中经常使用Tmux 实现分屏，windows 系统中可以使用ConEmu 实现类似功能。

ConEmu 是个可以集中托管各种终端窗口的控制台应用程序，通过它你可以在一个应用中同时使用cmd，Git Bash，PowerShell等多个应用程序。当然也可以间接实现同一个控制台应用的分屏操作。

安装

直接下载安装包，双击安装即可。
